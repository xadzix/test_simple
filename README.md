# СметаХаб

Веб-приложение для импорта прайс-листов поставщиков и смет, ведения каталога и автоматического сопоставления позиций.

## Что реализовано

- CRUD поставщиков с поиском по названию и ИНН, валютой и историей прайс-листов.
- Импорт `.xlsx` и `.xls`: выбор листа, строки заголовка, маппинг колонок и превью первых строк.
- Фоновый разбор файлов в Celery с прогрессом, итоговым статусом и сообщением об ошибке.
- Каталог товаров с группами и привязкой предложений поставщиков по точному артикулу.
- Проекты и несколько смет внутри каждого проекта.
- Гибкий импорт смет: наименование, артикул, единица, количество, материалы и монтаж.
- Автосопоставление с объяснимой уверенностью: точный артикул или семантическое сходство OpenAI Embeddings.
- Ручное исправление сопоставления и явный статус «Без соответствия».
- Зелёная/красная индикация уверенности, суммы и сводные показатели сметы.
- Адаптивный React-интерфейс и полный запуск через Docker Compose.

## Быстрый запуск


Создайте `.env` из `.env.example` и укажите `OPENAI_API_KEY`. Без ключа импорт продолжит работать, но позиции без артикула останутся без соответствия.

```bash
docker compose up --build
```

После успешного старта приложение доступно на [http://localhost:8080](http://localhost:8080), API — на `http://localhost:8080/api/`.

Остановка:

```bash
docker compose down
```

Удаление вместе с данными PostgreSQL и загруженными файлами:

```bash
docker compose down -v
```


## Проверка

Backend-тесты:

```bash
docker compose run --rm backend python manage.py test
```

Frontend production build выполняется прямо при сборке Docker-образа.
